REDESIGN GUIDE 
SUSANTEDIT: Redesign Handover Guide
This guide ensures that any agent working on the SUSANTEDIT project adheres to the "1st Place" quality standards established during the Cyber-Premium redesign.

🧬 Design DNA
Colors: Use --primary (#e63946) for actions and --secondary (#a855f7) for highlights. Backgrounds MUST be deep black (#030303).
Glassmorphism: Always apply the .glass class for panels. Use backdrop-filter: blur(20px) and a subtle border: 1px solid rgba(255,255,255,0.1).
Glows: Use filter: drop-shadow(0 0 10px var(--primary)) for icons and active elements.
🕹️ Component Instructions
1. 3D Elements (Three.js / R3F)
Scene3D: Keep this as the background layer. It contains the stars and nebula.
EarthGlobe: Used in the Bento Grid. Do not increase resolution (markers) too high to maintain performance.
Floating Assets: When adding new 3D models, use the useGLTF hook from @react-three/drei and wrap them in <Float />.
2. Animations (Framer Motion)
Transitions: Use initial={{ opacity: 0, y: 20 }} animate={{ opacity: 1, y: 0 }} for staggered entrances.
Magnetic Buttons: Wrap buttons in a motion div that tracks cursor offset.
Scroll reveal: Use whileInView for bento grid items to make them pop as the user scrolls.
3. Page-Specific Tasks
LandingPage: Hero needs a 3D Diamond asset. The Bento Grid should have a 3D Calendar cell.21
Dashboards: Sidebar should be glassmorphic with a slide-in animation. Stat cards should have a shimmer effect on load.
Auth: Use a central glass card with a "Pulsing Core" shader background.
⚠️ Important Notes
Performance: Always use <Suspense /> when loading 3D assets to prevent UI blocking.
Responsiveness: Check mobile views. 3D elements should often be disabled or simplified on mobile to save battery and maintain FPS.
